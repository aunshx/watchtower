# Watchtower Dashboard

The read-only review UI for the Watchtower agent.

Stack: React + Vite + TypeScript + Tailwind. Static export, deployed to GitHub Pages.

## Run locally

```bash
cd web
npm install
npm run dev
```

## Data source

Reads from `public/data.json`, which is generated by `../scripts/build_dashboard_data.py` from the most recent agent run.

## Deploy

Pushes to main that change anything under `web/` trigger the deploy workflow at `.github/workflows/deploy.yml`.

Live at: https://aunshx.github.io/watchtower/
